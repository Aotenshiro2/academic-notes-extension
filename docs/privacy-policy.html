Politique de Confidentialité

Trading Notes by AOKnowledge

Dernière mise à jour : 12 janvier 2026

En résumé : Cette extension ne collecte, ne stocke et ne transmet AUCUNE donnée personnelle. Toutes vos notes restent privées et locales sur votre appareil.

1. Données collectées

Aucune donnée personnelle n'est collectée par cette extension.

L'extension Trading Notes by AOKnowledge :

  • ❌ Ne collecte aucune information personnelle
  • ❌ Ne transmet aucune donnée vers des serveurs externes
  • ❌ N'utilise aucun service d'analytics ou de tracking
  • ❌ Ne partage aucune information avec des tiers

2. Stockage local

Toutes vos notes sont stockées exclusivement en local dans votre navigateur :

  • ✅ Vos notes restent sur votre appareil
  • ✅ Aucune synchronisation cloud
  • ✅ Aucun accès externe à vos données
  • ✅ Contrôle total de vos informations

3. Permissions utilisées

L'extension demande certaines permissions uniquement pour son fonctionnement :

  • sidePanel : Afficher l'interface dans le panneau latéral
  • storage : Sauvegarder vos notes localement
  • activeTab : Obtenir le titre/URL de la page et permettre les captures d'écran
  • tabs : Accéder aux informations des onglets
  • contextMenus : Menu contextuel (clic droit)

4. Utilisation des données

Les données que vous créez (vos notes, captures d'écran) sont utilisées uniquement pour :

  • Afficher vos notes dans l'interface
  • Organiser vos contenus par page web
  • Rechercher dans vos propres notes

5. Sécurité

Vos données sont sécurisées car elles :

  • Restent dans le stockage local de votre navigateur
  • Ne transitent jamais par Internet
  • Ne sont accessibles qu'à vous via l'extension

6. Suppression des données

Vous pouvez supprimer toutes vos données à tout moment :

  • En supprimant l'extension
  • En effaçant les données de navigation de votre navigateur
  • En supprimant individuellement vos notes dans l'interface

7. Modifications

Cette politique peut être mise à jour. Toute modification sera communiquée via les mises à jour de l'extension.

8. Conformité

Cette extension respecte :

  • Le Règlement du programme Chrome Web Store pour les développeurs
  • Les principes de confidentialité by design
  • La réglementation RGPD (aucune donnée collectée)
